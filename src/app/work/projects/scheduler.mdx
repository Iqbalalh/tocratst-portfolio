## Overview

Penjadwalan Unila is a fully featured scheduling system designed to streamline academic scheduling operations. It provides CRUD functionality for key academic entities such as lecturers, courses, departments, faculties, and more. Beyond manual management, the system includes an automated schedule generation feature powered by an Integer Linear Programming (ILP) algorithm for optimal timetable creation.

## Key Features

- **Comprehensive Management**: Create, update, and manage lecturers, courses, faculties, departments, rooms, and other essential scheduling entities.
- **Automated Schedule Generation**: Uses an ILP model to generate optimized schedules that minimize conflicts and improve allocation efficiency.
- **Structured Workflow**: A well-structured interface enabling administrators to maintain academic data and generate timetables easily.

## Technologies Used

- **Next.js**: For building the main service, handling UI, routing, and server-side logic.
- **FastAPI**: For the automation service implementing the Integer Linear Programming model.
- **PostgreSQL**: As the main relational database to store structured academic data.
- **Tailwind CSS**: For building a clean and responsive interface.

## Outcome

Penjadwalan Unila provides Universitas Lampung with a streamlined and automated approach to academic scheduling. The system reduces manual workload, minimizes scheduling conflicts, and ensures efficient resource allocation through optimization algorithms. It demonstrates strong integration of full-stack development, mathematical modeling, and database engineering.
