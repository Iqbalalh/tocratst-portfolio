## Overview

Geopusaka is an internal web application built for Yayasan Pusaka KAI using Next.js. It functions as a map-based platform for visualizing the distribution and locations of Keluarga Asuh under Yayasan Pusaka KAI, fully integrated with the Nayapusaka Backend Service API.

## Key Features

- **Interactive Map Visualization**: Implemented a dynamic map interface using Leaflet to display UMKM locations, home distributions, and regional boundaries with smooth filtering and clustering capabilities.
- **Backend Service Integration**: Connected Geopusaka with the Nayapusaka API for authentication, data fetching, and realtime updates, ensuring consistency across the organization’s internal systems.
- **Responsive UI/UX**: Designed and developed a clean, responsive interface optimized for both desktop and mobile workflows, improving usability for field staff and administrators.

## Technologies Used

- **Next.js**: For fast, scalable, and optimized web app development.
- **Leaflet**: For rendering interactive geospatial maps.
- **Tailwind**: For flexible, component-driven styling.

## Outcome

Geopusaka now serves as a central visualization tool for Yayasan Pusaka KAI. It has improved data accessibility and provided staff with a modern, intuitive platform for visualizing UMKM and regional information.
