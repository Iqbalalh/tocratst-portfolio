## Overview

GEOPUSAKA is a map-driven internal application built with Next.js, designed to streamline the visualization of UMKM, household, and regional data. The platform integrates seamlessly with the NAYAPUSAKA Backend Service API, enabling real-time data synchronization and operational efficiency for Yayasan Pusaka KAI.

## Key Features

- **Interactive Map Visualization**: Implemented a dynamic map interface using Leaflet to display UMKM locations, home distributions, and regional boundaries with smooth filtering and clustering capabilities.
- **Backend Service Integration**: Connected GEOPUSAKA with the NAYAPUSAKA API for authentication, data fetching, and real-time updates, ensuring consistency across the organization’s internal systems.
- **Responsive UI/UX**: Designed and developed a clean, responsive interface optimized for both desktop and mobile workflows, improving usability for field staff and administrators.

## Technologies Used

- **Next.js**: For fast, scalable, and optimized front-end development.
- **Leaflet**: For rendering interactive geospatial maps.
- **TypeScript**: For stronger type safety and cleaner code maintenance.
- **Tailwind**: For flexible, component-driven styling.

## Outcome

GEOPUSAKA now serves as a central visualization tool for Yayasan Pusaka KAI. It has improved data accessibility, reduced manual administrative work, and provided staff with a modern, intuitive platform for managing UMKM and regional information. This project demonstrates a strong combination of mapping technology, front-end engineering, and API integration.
