## Overview

Akademikerja.id is a web platform developed to streamline the connection between job seekers, training institutions, and employers. Built with Next.js and MongoDB, the platform provides a structured system for account creation, document management, and job applications, enabling users to explore both domestic and international employment opportunities with ease.

## Key Features

- **User Account System**: Developed secure account creation, authentication, and profile management features for job seekers, institutions, and employers.
- **Document Management**: Implemented document upload functionality for CVs, certificates, and administrative requirements using a scalable storage approach.
- **Job Listings Integration**: Added support for domestic and international job listings with filtering and structured data display for better discoverability.
- **Application Workflow**: Enabled job seekers to apply directly through the platform, with institutions and employers able to manage candidate submissions efficiently.
- **Admin Tools**: Built internal tools for administrators to verify documents, manage content, and monitor platform activity.

## Technologies Used

- **Next.js**: For building fast, modern, and SEO-friendly front-end applications.
- **MongoDB**: For handling dynamic user and job-related data with flexible schema design.
- **TypeScript**: For safer, maintainable, and scalable development.
- **Tailwind CSS**: For creating a clean and responsive UI.

## Outcome

Akademikerja.id successfully provides an integrated ecosystem for job discovery, training access, and employment matching. The platform improves user experience for job seekers, streamlines workflows for institutions, and expands reach for employers. This project highlights the effective combination of full-stack development, scalable storage, and well-structured user experience design.
