## Overview

The Tribratanews clone is a police news portal built to deliver fast, accurate, and trustworthy public information. The platform provides structured news browsing, detailed article pages, and a secure backend dashboard for managing published content.

## Key Features

- **News Browsing & Reading**: A clean, intuitive interface for exploring the latest police news, categories, and article details.
- **Admin Dashboard**: A role-based CMS built for creating, editing, and managing articles, categories, and media content.
- **Responsive Layout**: Optimized for desktop and mobile, ensuring reliable access for all users.

## Technologies Used

- **Laravel**: Backend framework for API development, authentication, and content management logic.
- **React.js**: For a dynamic and responsive user experience.
- **PostgreSQL**: As a reliable and scalable database solution.

## Outcome

The Tribratanews clone successfully replicates a professional-grade police news portal with modern tooling, strong data integrity, and a maintainable architecture. It provides a robust foundation for scalable content publication and efficient administrative workflows.
